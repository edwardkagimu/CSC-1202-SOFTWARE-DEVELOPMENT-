**#INTERNSHIP LOGGING AND EVALUATION SYSTEM**



**## User\_stories**



**### Student Intern** 

    Us-01

&nbsp;    As a student intern, I want to log into the system to access my internship dashboard.

&nbsp;   Us-02

&nbsp;    As a student intern, I want to want to view my internship placement details, so I that I know where am assigned.

&nbsp;   Us-03

&nbsp;    As a student intern, I want to edit my log before submission so that I can correct my mistakes.

&nbsp;   Us-04

&nbsp;    As a student intern, I want to submit weekly internship logs so that supervisors can review my work.

&nbsp;   Us-05

&nbsp;    As a student intern, I want to view feedback from supervisors so that I can improve my performance

&nbsp;   Us-06

&nbsp;    As a student intern, I want to see whether my log is reviewed or approved so that I know my progress.



**### Workplace-supervisor**

    Us-07

&nbsp;    As a workplace-supervisor, I want to review intern logs so that I can verify their activities.

&nbsp;  Us-08

&nbsp;    As a workplace-supervisor, I want to approve or reject logs so that only valid work is approved.

&nbsp;  Us-09

&nbsp;    As a workplace-supervisor, I want to leave comments so that interns receive feedback.

&nbsp;  Us-10

&nbsp;    As a workplace-supervisor, I want to submit an internship evaluation , so that the student can be graded.



**### Academic-supervisor**

    Us-11

&nbsp;    As an academic-supervisor, I want to view student weekly logs, so that I can monitor progress.

&nbsp;   Us-12

&nbsp;    As an academic-supervisor, I want to see workplace-supervisor comments so that I understand student performance.

&nbsp;   Us-13

&nbsp;    As an academic-supervisor, I want to evaluate the intern academically, so that  final grade can be calculated.



**### Internship-administrator**

&nbsp;   Us-14

&nbsp;   Us-16

&nbsp;     As an internship-administrator, I want to view internship progress, so that I can monitor the program.       

&nbsp;   Us-17

&nbsp;     As an internship-administrator, I want to generate internship reports, so that the institution can evaluate outcomes.
